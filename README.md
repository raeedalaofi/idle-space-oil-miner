# Idle Space Oil Miner — Legal

Public privacy policy page for the Android game **Idle Space Oil Miner**.

## Live URL (GitHub Pages)

After enabling Pages, the policy will be available at:

**https://raeedalaofi.github.io/idle-space-oil-miner/**

Use this URL in Google Play Console → **App content** → **Privacy policy**.

## Setup GitHub Pages

1. Push this repository to GitHub.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose branch **main** (or **master**) and folder **/ (root)**.
5. Save. The site may take a few minutes to go live.
